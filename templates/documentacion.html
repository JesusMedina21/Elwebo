Documentacion de la API del Sistema Nacional de Orquestas y Coros Infantiles y Juveniles de Venezuela


Metodo GET para consumir la API (obviamente cualquier persona puede usar este metodo)


GET https://api01.pythonanywhere.com/api/Instrumentos/

METODO POST para agregar mas Instrumentos (solo un administrador puede usar este metodo)


POST https://api01.pythonanywhere.com/api/Instrumentos/

Campos obligatorios

Nombre_Del_Instrumento Requerido
Familia_Del_Instrumento Requerido
Foto Requerido

Metodo PUT para actualizar un Instrumento (solo un administrador puede usar este metodo)


PUT https://api01.pythonanywhere.com/api/Instrumentos/{id}/

Campos obligatorios

Nombre_Del_Instrumento Requerido
Familia_Del_Instrumento Requerido
Foto Requerido

Metodo DELETE para eliminar un Instrumento (solo un administrador puede usar este metodo)


DELETE https://api01.pythonanywhere.com/api/Instrumentos/{id}/

Campo Obligatorio

id Requerido
